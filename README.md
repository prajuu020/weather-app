🌦️ Weather App

A simple and responsive weather application built using React that fetches real-time weather data using the OpenWeather API.

🚀 Features

- Search weather by city name
- Displays temperature, humidity, and weather condition
- Real-time API data fetching
- Clean and responsive UI

🛠️ Tech Stack

- React.js
- JavaScript
- OpenWeather API
- CSS

🔗 Live Demo

https://weather-app-qk0c.onrender.com

📂 GitHub Repository

https://github.com/prajuu020/weather-app

⚙️ Setup Instructions

1. Clone the repository
2. Run "npm install"
3. Run "npm run dev"

🔐 Environment Variables

Create a ".env" file and add:
VITE_API_KEY=your_api_key
